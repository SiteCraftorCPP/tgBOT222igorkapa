# Как работать с логами бота

## Способ 1: Запуск с выводом в консоль (видно в реальном времени)
```bash
python bot.py
```
Или просто запустите `run_bot.bat`

## Способ 2: Запуск с логированием в файл
```bash
python bot.py > bot.log 2>&1
```
Или запустите `run_bot_log.bat`

## Способ 3: Просмотр логов в реальном времени (tail)
После запуска бота с логированием в файл, откройте новое окно терминала и:
```bash
view_log.bat
```
Или в PowerShell:
```powershell
Get-Content bot.log -Wait -Tail 50
```

## Структура логов

- `[NEW]` - новая пара добавлена
- `[INIT]` - пара инициализирована (warmup завершён)
- `[WARMUP]` - пара в режиме ожидания инициализации
- `[UP]` - цена выросла, обновлён локальный максимум
- `[RESET]` - состояние пары сброшено
- `[!!!]` - **СИГНАЛ** - обнаружено падение цены
- `[SIGNALS]` - отправлено сигналов в Telegram
- `[SCAN]` - статистика сканирования пар за цикл
- `[ERROR]` - ошибки при обработке

## Примеры полезных команд

Посмотреть последние 100 строк:
```powershell
Get-Content bot.log -Tail 100
```

Найти все сигналы:
```powershell
Select-String -Path bot.log -Pattern "\[!!!\]"
```

Найти ошибки:
```powershell
Select-String -Path bot.log -Pattern "\[ERROR\]"
```

Посмотреть статистику сканирования:
```powershell
Select-String -Path bot.log -Pattern "\[SCAN\]"
```
